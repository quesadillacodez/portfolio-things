# Hadi Qusyairi Portfolio

A responsive, single-page React portfolio focused on practical software, analytics, and operational problem solving.

## Featured work

- PulseOps EMS Command Center
- Global Food Insecurity Dashboard
- FairPrice Shopping Simulation
- Clinic Digitalisation Concept

## Run locally

```bash
pnpm install
pnpm run dev
```

## Production build

```bash
pnpm run build
```

Project content is stored in `src/data/projects.js`, making it straightforward to add repository and demo links as they become available.
